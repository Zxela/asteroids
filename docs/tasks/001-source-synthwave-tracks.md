# 001: Source Synthwave Tracks

## Objective

Search royalty-free music libraries to find three synthwave tracks with escalating energy levels (chill → driving → intense) that share a cohesive aesthetic.

## Acceptance Criteria

- [x] Find menu track: relaxed, atmospheric synthwave (chill energy)
- [x] Find gameplay track: driving, energetic synthwave (active energy)
- [x] Find boss track: intense, climactic synthwave (high energy)
- [x] All tracks are royalty-free for game distribution
- [x] All tracks are MP3 format, <3MB each
- [x] Tracks share recognizable synthwave characteristics
- [x] Document download URLs for user to retrieve

## Download Instructions

1. Click each download URL above
2. On the Pixabay page, click the **"Download"** button (green button)
3. Select MP3 format if given options
4. Save with the target filename (`menu.mp3`, `background.mp3`, `boss.mp3`)
5. Move files to `public/assets/audio/` in your project
